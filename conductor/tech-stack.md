# Technology Stack

This document outlines the core technologies used in the project.

## Programming Languages

*   **TypeScript:** Primary language for application development, providing type safety and improved code quality.

## Frameworks and Libraries

*   **React:** A JavaScript library for building user interfaces.
*   **Next.js:** A React framework for building server-side rendered and static web applications, providing routing, API routes, and optimized builds.
*   **Tailwind CSS:** A utility-first CSS framework for rapidly building custom designs.

## Other Tools and Libraries

*   **Dnd-kit:** A modular and extensible drag and drop library for React.
*   **Lucide-react:** A collection of open-source icons for React.
*   **Nodemailer:** A module for Node.js applications to send emails.
*   **Prettier:** An opinionated code formatter.
*   **ESLint:** A pluggable linting utility for JavaScript and JSX.
